🌱 IoT-Based Air Pollution Monitoring System
📌 Overview
The IoT-Based Air Pollution Monitoring System is a smart environmental monitoring project designed to measure air quality in real time using IoT technology. It detects harmful gases and pollution levels using sensors and provides instant feedback through display and alert systems.
This project helps in understanding environmental conditions and supports the concept of smart cities and environmental safety awareness.
🎯 Objective
To design a low-cost, efficient, and real-time air quality monitoring system that can:
Detect harmful gases in the environment
Display live air quality data
Trigger alerts when pollution exceeds safe limits
Enable IoT-based monitoring for remote access (if extended with cloud)
🛠️ Hardware Components
NodeMCU (ESP8266) – Microcontroller with Wi-Fi
MQ135 Gas Sensor – Air quality detection
I2C LCD Display – Real-time data display
LED Indicators – Visual alert system
Buzzer – Sound alert system
Power Supply
⚙️ Software & Tools
Arduino IDE
Embedded C / C++
IoT Communication (Wi-Fi via ESP8266)
Serial Monitoring
🔄 Working Principle
MQ135 sensor continuously measures air pollutants (CO₂, smoke, ammonia, etc.).
Sensor data is sent to NodeMCU for processing.
The system calculates air quality levels in real time.
Results are displayed on LCD.
If pollution crosses threshold:
LED turns ON
Buzzer activates for alert
🚨 Features
🌍 Real-time air quality monitoring
